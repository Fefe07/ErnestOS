# ErnestOS

ErnestOS is an Operating System developed during an OS course at the ENS.

## Building

In order to compile our OS, you will need a i686 cross-compiler (see https://wiki.osdev.org/GCC_Cross-Compiler)  
You must indicate the path of your cross-compiler with 

```export PATH="/path/to/file/foo/bar:$PATH"```

You can then build it with `make`, and run with `make run`.

## Usage 

Type `help` for command list, and `help foo` for more precisions.
NB : the OS is in French so far.
